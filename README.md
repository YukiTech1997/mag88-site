# mag88-site
Codejumpのmag88-siteの模写コーディングをしました。(著作権フリーです。)
デザイン引用元　URL https://code-jump.com/html/
使用素材：サイト配布のフリー素材を使用

## 使用技術
- HTML / CSS(SCSS)
- FLOCSSとBEM記法
- Viteで環境構築

## 見どころ・工夫した点
- FLOCSSとBEMを組み合わせて保守性の高いコーディングを心がけました。

## 学んだこと
- FLOCSSのファイル分割の方法
- FLOCSSとBEMの組み合わせでクラス名を命名する
